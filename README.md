# Подарок мечты

Веб-сервис для «угадывания» желаемого подарка. Пользователь проходит опрос (праздник, категория, бюджет, стиль, эмоции и т.д.), получает конкретную рекомендацию подарка и может прикрепить фото-примеры. Результаты отправляются в Telegram.

## Возможности

- **Два сценария:** «Хочу получить подарок» (пройти опрос самому) и «Узнать, что хочет другой» (получить ссылку, отправить близкому — результаты придут вам в Telegram).
- **Детальный опрос:** праздник, возраст, хобби (мультивыбор), категория, бюджет, стиль, эмоция, цвет, размер, упаковка, пожелания, конкретная идея.
- **Конкретная рекомендация подарка** на основе ответов (не перечисление ответов, а реальное предложение).
- **Загрузка фото-примеров** (до 10 штук) — отправляются в Telegram вместе с результатами.
- **Telegram-бот:** пользователь пишет `/start`, регистрируется, затем на сайте получает ссылку. Результаты + фото приходят в чат.
- **Ссылки действуют 30 дней.**

## Локальный запуск

```bash
git clone <url> && cd web-gift
npm run install:all
```

Создайте `.env` в корне:

```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
PORT=3000
```

Соберите фронтенд и запустите:

```bash
npm run build
npm start
```

Сайт: http://localhost:3000

## Деплой

### Render.com (рекомендуется)

1. Создайте репозиторий на GitHub и запушьте проект.
2. На [render.com](https://render.com) создайте **Web Service**, подключите репозиторий.
3. Настройки:
   - **Build Command:** `npm install && npm run build`
   - **Start Command:** `node server.js`
4. В разделе **Environment** добавьте переменную `TELEGRAM_BOT_TOKEN`.
5. Деплой произойдёт автоматически.

Файл `render.yaml` уже включён — можно использовать Blueprint для автоматической настройки.

### Railway.app

1. Подключите репозиторий на [railway.app](https://railway.app).
2. Railway автоматически определит Node.js проект.
3. Добавьте переменную окружения `TELEGRAM_BOT_TOKEN`.
4. Деплой автоматический — `postinstall` скрипт соберёт фронтенд.

### Docker

```bash
docker build -t web-gift .
docker run -p 3000:3000 -e TELEGRAM_BOT_TOKEN=ваш_токен web-gift
```

### Heroku

```bash
heroku create
heroku config:set TELEGRAM_BOT_TOKEN=ваш_токен
git push heroku main
```

`Procfile` уже включён.

## Переменные окружения

| Переменная | Обязательная | Описание |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | Да | Токен Telegram-бота от @BotFather |
| `PORT` | Нет | Порт сервера (по умолчанию 3000) |

## Безопасность

- Токен бота только в `.env` — файл в `.gitignore`.
- Загруженные фото удаляются с сервера сразу после отправки в Telegram.
